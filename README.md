## **Project Structure**
The project Structure is as follows:

Prediction_Of_Heart_Failure/
│
├── data/
|     ├── processed/            # Folder for processed data
|     ├── raw/                  # Folder for raw data
|     ├── SQL/                  # Folder for tables|
